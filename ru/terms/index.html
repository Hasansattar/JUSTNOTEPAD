Сервис Justnotepad.com использует «Google Analytics» для того, чтобы собирать, мониторить и анализировать «Лог данные» в целях повышения функциональности Сервиса Justnotepad.com и удобства для пользователя, и для того, чтобы лучше приспособить Сервис к потребностям посетителей. Политика «Локального хранилища»

Сервис Justnotepad.com использует технологию «Локального хранилища» в целях повышения функциональности и удобства использования Сервиса. «Локальное хранилище» — это расположение во временной памяти (как данные ключ-значение) на устройстве Пользователя, которое доступно для браузера Пользователя. «Локальное хранилище» может оставаться на устройстве Пользователя, даже после того, как Пользователь закрыл свой браузер, в зависимости от настроек браузера. «Локальное хранилище» хранится согласно настройкам безопасности браузера, которые могут быть установлены Пользователем. Сервис Justnotepad.com использует «Локальное хранилище» главным образом, чтобы предоставить Пользователю функционал автоматического сохранения текста (черновика) на устройстве пользователя, для того, чтобы продолжить использовать текст (черновик) позже, даже после того, как Пользователь закрыл браузер (если это возможно технологически и поддерживается браузером). Браузеры, которые установлены в режим «Не запоминать историю» или «Приватный режим» будет использовать «Локальное хранилище» пока браузер не будет закрыт. В этом случае, Пользователь не сможет возобновить (восстановить) ранее введенный текст (черновик) на Сервисе Justnotepad.com после перезапуска браузера, потому что «Локальное хранилище» будет удалено с устройства Пользователя.
